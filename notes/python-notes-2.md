# Listas com split()

```py
a = input().split(' ')
```

entrada:
```py
5 4 1 2 3
```

## Acessando elementos

A posição é passada através dos conchetes. Pode ser usado tanto pra consultar quanto para alterar.

```py
lista = input().split()

#imprime o primeiro elemento
print(lista[0]) 
```
