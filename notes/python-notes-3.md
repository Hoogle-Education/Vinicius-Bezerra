# Estruturas de Repetição

```
1. inicialização
2. verificação
3. atualização
```

## Estrutura `while`

```py
numero = 11

while numero <= 10 :
  print(numero)
```

## Estrutura `for`

```py
for numero in range(10):
  print(numero)
```
